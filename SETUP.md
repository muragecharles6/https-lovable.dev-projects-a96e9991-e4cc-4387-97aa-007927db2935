# Setup Instructions for Tenda F3 WiFi Client Management Android App

## Prerequisites

Before you begin, ensure you have the following installed:
1. **Java JDK** - version X or higher
2. **Android Studio** - version X.X or higher
3. **Gradle** - version X.X or higher
4. Any other dependencies that may be required.

## Cloning the Repository

1. Clone the repository using Git:

   ```bash
   git clone https://github.com/muragecharles6/https-lovable.dev-projects-a96e9991-e4cc-4387-97aa-007927db2935.git
   ```

2. Navigate into the project directory:

   ```bash
   cd dev-projects-a96e9991-e4cc-4387-97aa-007927db2935
   ```

## Building the Project

1. Open the project in Android Studio.
2. Wait for the Gradle build to finish.
3. Ensure all dependencies are correctly downloaded.

## Running the App

1. Connect your Android device or start an emulator.
2. Click on the 'Run' button in Android Studio or use the following command:

   ```bash
   ./gradlew installDebug
   ```

3. The app should now be installed on your connected device or emulator.

## Troubleshooting

- **Build Issues**: Make sure all prerequisites are installed and environment variables are correctly configured.
- **Permission Issues**: Check your device settings and application permissions if you encounter issues while running the app.
